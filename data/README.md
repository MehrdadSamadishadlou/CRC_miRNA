# Data files

Large data files are intentionally excluded from version control.

## Raw data

Place the following files in `data/raw/`:

```text
no_cancer_expression.txt
CR_expression.txt
```

These are the non-cancer and colorectal-cancer expression matrices used by the supplied R script.

## External annotation data

Place the TarBase file in `data/external/`:

```text
Homo_sapiens_TarBase-v9.tsv
```

## Files used to create the training/test split

Place the following files in `data/processed/`:

```text
Expression_all.csv
exp_all_normal.txt
```

Run `notebooks/00_train_test_split.ipynb` to create:

```text
train_set.csv
test_set.csv
```

The split uses a stratified 75:25 partition with `random_state=1`.

## Files generated by the R script

The R script creates or expects the following files:

```text
CRC_all_data.csv
Expression_all.csv
train_normal.txt
train_exp_selected_mirs.txt
test_exp_selected_mirs.txt
```
